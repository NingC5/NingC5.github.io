网站防护：Cloudflare

本文最后更新于 2025年6月18日 晚上

Cloudflare 作为反代理服务器能够增强网站安全性，如果你使用 Cloudflare 代理你的网站。

Tip：在 Cloudflare 上添加 DNS 记录时勾选小黄云，就是使用 Cloudflare 代理你的网站。如果你不想更改主域名的 NS 为 Cloudflare，那就只能在 Cloudflare 上牺牲备用域名，然后备用域名 CNAME 主域名。

代理服务器是请求发起方与目标资源之间的中介。正向与反向只是概念上的差异——架设者与架设目的。正向代理服务器着重于监管（保护|控制）前者，由监管者架设；反向代理服务器聚焦于保护后者所在的 Web 服务器，由站主架设。

Bot Fight Mode

Bot Fight Mode

Bot Fight Mode（自动程序攻击模式）是 free plan 提供的一项服务。它提供强大防护的同时，也存在一些缺陷。

付费 plan 提供的是优美的 super 模式。

本节只简单地说一下「流量」和「WAF 自定义规则」两方面，详情见官方文档。

该模式虽然能拦截部分恶意流量，但响应这些恶流所发起的质询仍然会消耗你的流量。该模式的质询拦截并不是一个好处理，理想方式应当是直接阻止恶流。

security rules 是在规则集引擎中实现的，能直接阻止恶流。但 Bot Fight Mode 却不归属于规则集引擎，启用该模式，匹配到恶流以阻止其访问的安全规则并不会生效。

如果不是很清楚如何防护你的网址，推荐开启该模式。（自行决定，前期开启，or 付费 plan）
